To Run Via Double Click:

	1.Navigate to "JMRunnable" Folder
	2.Double Click "JavaMatic.sh" (Java Matic Shell Script)
	
	Important Note: Make sure "JavaMatic.sh" and "JavaMaticRunnable.jar" remain in the same folder

To Run Via Command Line:

	1.Open Command Line and Navigate to "JMRunnable" folder
	2.Enter Command: 
		java -jar JavaMaticRunnable.jar
